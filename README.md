# Контрольная работа №2 по фронтенд/бэкенд разработке
## [Практическая работа №7-12 (см. комиты)](./PR7-8)
### Практическое занятие №7
Базовая аутентификация. Выполнено создание серверного приложения на Node.js. Реализованы механизмы регистрации и входа пользователей. Внедрено хеширование паролей с помощью библиотеки bcrypt для обеспечения безопасности данных.

### Практическое занятие №8
Работа с JWT. Осуществлена доработка сервера для выдачи access-токенов (JWT) при успешной авторизации. Добавлен защищенный маршрут /api/auth/me для получения информации о текущем пользователе.

### Практическое занятие №9
Refresh-токены. Реализован механизм обновления сессий через refresh-токены. Это позволило обеспечить автоматическое продление доступа без необходимости повторного ввода учетных данных.

### Практическое занятие №10
Фронтенд и хранение токенов. Разработана клиентская часть приложения (на базе React.js или Vue.js). Настроено безопасное хранение токенов в браузере и реализован интерфейс для управления списком товаров.

### Практическое занятие №11
Система ролей (RBAC). Внедрена модель контроля доступа на основе ролей. Установлено разграничение прав для ролей «Администратор», «Продавец» и «Пользователь», как на уровне API-запросов, так и в пользовательском интерфейсе.
